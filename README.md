Planet Simulator (PlaSim) is .. dimensional with .. outputs. Information about all the variables can be found in the guide.
You can run it on the HPC or your local machine (obviously the HPC would be faster). If you run it on your local machine, you have the option of generating the GUI while it runs - a series of graphs which look like the screenshot below. Of course this is not practical in long simulations.
The steps to run a basic simulation are simple:
1. clone the git repository from ..
2. load modules
3. configure
4. launch most

You then see a GUI, which will allow you to select several options of your run. When running on the HPC, you must untick run with GUI, since it's not supported. 
When launching PLASIM, you can instead run it with a bash script.
